# 🧭 湾趣地图 · 大湾区亲子活动指南

一张覆盖**粤港澳大湾区 11 城**的亲子活动地图网页应用：广州、深圳、珠海、佛山、惠州、东莞、中山、江门、肇庆、香港、澳门。

公园、海滩、博物馆、玩水、采摘、露营……周末去哪儿遛娃，打开地图就知道。

## ✨ 功能

- 🗺 **交互地图**：270+ 亲子地点按分类彩色标注，自动聚合，点击查看详情
- 🏷 **13 个分类**：游乐场 / 室内乐园 / 公园绿地 / 图书馆 / 博物馆·科普 / 动物·海洋 / 海滩 / 玩水 / 徒步·自然 / 露营 / 采摘农场 / 主题乐园 / 商场亲子
- ⚲ **组合筛选**：按分类、城市、免费/收费、室内/户外、幼儿友好任意组合（比如雨天方案 =「免费 + 室内」）
- 🔍 **即时搜索**：按名称、城市、关键词、标签搜索
- 🎯 **附近活动**：一键定位，按距离排序显示身边的活动
- ☰ **列表视图**：地图 / 列表双视图切换
- 📱 **移动端适配**：手机浏览器体验完整，筛选栏抽屉式收起
- 🚗 **一键导航**：详情卡片直达高德 / 百度 / Google 地图

## 🚀 上线（GitHub Pages，免费）

本项目是**纯静态网页**，无需服务器和构建步骤：

1. 进入仓库 **Settings → Pages**
2. **Source** 选择 **GitHub Actions**
3. 将本分支合并到 `main`（或在 Actions 页手动运行 *Deploy to GitHub Pages* 工作流）
4. 几分钟后即可通过 `https://<用户名>.github.io/family_activity_map/` 访问

也可以直接把整个目录拖到 [Netlify](https://app.netlify.com/drop) / Vercel，或双击 `index.html` 本地打开。

> 💡 若主要用户在中国内地，建议后续把域名部署到国内可稳定访问的托管（如 Vercel 自定义域名、腾讯云 COS 静态托管等），并考虑将地图底图换成高德/腾讯瓦片以提升加载速度。

## 🧱 技术栈

- 原生 HTML / CSS / JavaScript，零构建、零依赖安装
- [Leaflet](https://leafletjs.com/) + [Leaflet.markercluster](https://github.com/Leaflet/Leaflet.markercluster)（CDN 引入）
- 底图：CARTO Voyager（基于 OpenStreetMap，免费无需 API key）
- 数据：`js/data.js` 单文件维护，纯前端过滤

## 📝 数据维护

所有地点数据在 [`js/data.js`](js/data.js)，每条记录格式：

```js
{
  n: "深圳湾公园",        // 名称
  c: "park",             // 分类 id（见 js/app.js 中 CATEGORIES）
  city: "深圳",           // 城市
  lat: 22.517, lng: 113.947,  // WGS-84 坐标（近似）
  d: "滨海长廊，适合骑行…",    // 一句话简介
  age: "0-14",           // 适合年龄
  free: true,            // 是否免费
  io: "outdoor",         // indoor / outdoor / both
  addr: "南山区滨海大道",  // 地址
  tags: ["海景", "骑行"],  // 标签
  season: "6-8月",        // 可选：采摘/花期季节
}
```

欢迎提 PR 补充地点或纠错。
